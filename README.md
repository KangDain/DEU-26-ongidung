▶ 기본 git 명령어

# 주소 초기화
```
git remote remove origin
```

# 레포지토리 주소로 이동
```
git remote add origin https://github.com/KangDain/DEU-26-ongidung.git
```

# 넣고자 하는 branch 이름으로 이동
```
git checkout -b (branch 이름)
```

# push (레포지토리에 올리기)
```
git push -u origin (branch 이름)
```

▶ git에 있는 파일 가져오기 (아예 새로 가져올 떄)
```

git checkout (가져올 branch 이름)
```

▶ 이미 있는 파일에 업데이트
